# Core Linux Commands Cheat Sheet

## Navigation

- `pwd` → show current directory  
- `ls` → list files  
- `cd` → change directory  

## File Operations

- `touch file` → create file  
- `mkdir dir` → create directory  
- `cp` → copy  
- `mv` → move/rename  
- `rm` → delete  

## Viewing Content

- `cat` → show file  
- `less` → scroll view  
- `head` / `tail` → partial view  

## Help & Documentation

- `man command`  
- `command --help`
